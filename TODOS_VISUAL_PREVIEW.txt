KEY FEATURES:

✨ Newer Tasks First
   └─ New tasks appear at top of list automatically

🔍 Real-Time Search
   └─ Type in search box, tasks filter instantly
   └─ Case-insensitive, searches title only

📋 Status Filter
   └─ All Tasks (default)
   └─ Pending (incomplete)
   └─ Completed (finished)

🎯 Priority Filter
   └─ All Priorities (default)
   └─ High Priority (red)
   └─ Medium Priority (yellow)
   └─ Low Priority (green)

🔄 Combined Filters
   └─ All filters work together
   └─ Results update in real-time
   └─ Counter shows: "Showing X of Y tasks"
