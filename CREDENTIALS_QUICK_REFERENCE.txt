GMAIL - WHAT YOU NEED
================================================================================

GMAIL_EMAIL
  └─ Your Gmail address
     Example: john.doe@gmail.com

GMAIL_APP_PASSWORD
  └─ 16-character special password (NOT your regular Gmail password)
     Format: xxxx xxxx xxxx xxxx (with spaces)
     Example: abcd efgh ijkl mnop

HOW TO GET APP PASSWORD:
  1. Go to https://myaccount.google.com/
  2. Click Security
  3. Enable 2-Factor Authentication (if needed)
  4. Find "App passwords" section
  5. Select: Mail + Windows Computer
  6. Google gives you 16 chars → Copy it
  7. Paste in .env

WHAT IT DOES:
  ✅ Send emails
  ✅ Read emails
  ✅ Search emails
  ✅ Archive/delete
  ✅ Create drafts
  ✅ Add labels

TIME: ~10 minutes


================================================================================
WHATSAPP - WHAT YOU NEED
================================================================================

⭐ NO CREDENTIALS IN .ENV! ⭐

Instead: QR Code Login (saved locally)

HOW TO SETUP:
  1. Run: python -m src.automations.whatsapp_playwright
  2. Browser opens → Shows QR code
  3. Scan QR with phone WhatsApp camera
  4. Approve on phone
  5. Done! Session saved to ~/.whatsapp_session/

SESSION LOCATION:
  ~/.whatsapp_session/state.json

WHAT IT DOES:
  ✅ Send messages
  ✅ Send images/videos
  ✅ Send to multiple contacts
  ✅ React with emojis
  ✅ Get contact status

FEATURES:
  • No API keys needed
  • No passwords stored
  • Works with personal WhatsApp
  • Session persists between runs

TIME: ~5 minutes

SECURITY TIPS
================================================================================

✅ DO:
  • Generate new App Password for Gmail
  • Use only "Mail" permission on App Password
  • Store .env locally (NEVER commit to git)
  • Add .env to .gitignore
  • Keep credentials in environment variables

❌ DON'T:
  • Don't use your regular Gmail password
  • Don't share .env file
  • Don't commit credentials to git
  • Don't share WhatsApp session files
  • Don't hardcode passwords in code


================================================================================
TROUBLESHOOTING
================================================================================

GMAIL ERROR: "App passwords option not visible"
  Fix: Enable 2-Factor Authentication first

GMAIL ERROR: "IMAP authentication failed"
  Fix: Make sure it's App Password (not regular password)
       Spaces matter! Format: xxxx xxxx xxxx xxxx

WHATSAPP ERROR: "QR code not appearing"
  Fix: Kill chromium: pkill -f chromium
       Run again: python -m src.automations.whatsapp_playwright

WHATSAPP ERROR: "Session expired"
  Fix: rm -rf ~/.whatsapp_session/state.json
       Scan QR code again
